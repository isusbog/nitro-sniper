## Prerequisites

- npm >=6.0.0
- node >=14.0.0

## Install

```sh
npm i nitro-sniper
```

## Usage

```sh
git clone https://github.com/Zusier/nitro-sniper.git
```
```sh
cd nitro-sniper
```
edit *config.json* and put your discord token there (KEEP PRIVATE)
```sh
node index
```
